Работата е в екипи от по 5 ученика
Ръководителя на екипа: създава GitHub хранилище
Всички членове на екипа:
Клонират хранилището при себе си
Правят някакви промени локално
Предават (с commit) тези локални промени
Изпращат (с push) промените към отдалеченото хранилище
При конфликт: изтеглят промените от GitHub (fetch + merge)
